What is HTML?

HTML is a markup language that defines the basic structure of web pages. It simply gives browsers the content it needs to display and doesn’t care how the browser goes about displaying the content, as long as it’s displayed. In other words, HTML has a structural purpose, not a functional one.

HTML is made up of a unique syntax and alphabet as shown below:

Let's take a closer look at each component of an element:

  • HTML Tags. Most elements have an opening and closing tag. Opening tags precede the text and contain the element name enclosed by the brackets “<” and “>”. Closing tags are identical to opening tags, save for a backward slash that precedes the element name.

    • Some HTML tags examples:

    Open tag Close tag Description
    < h1 > </ h1 > The lagest heading
    < h6 > </ h6 > The smallest heading
    < p > </ p > This tag allows you to create paragraphs
    < b > </ b > Makes text bold
    < i > </ i > Makes text italic
  • HTML Atributes. While all HTML elements need tags, only some need attributes. An attribute provides additional information about the HTML element, and this information can be essential or non-essential. Most often used atributes is "href" - specifies the URL of the page the link goes to; "src" - specifies the path to the image to be displayed; "alt" - specifies the path to the image to be displayed; "style" -is used to add styles to an element, such as color, font, size, and more.
