What is CSS?

Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements.

CSS can be added to HTML documents in 3 ways:

  1. Inline - by using the style attribute inside HTML elements.

  2. Internal - code by using a style element in the <head> section.

  3. External - by using a <link/> element to link to an external CSS file.

The whole CSS structure is called a ruleset and looks like this:

Note the names of the individual parts:

  • Selector. This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled.

  • Declaration. This is a single rule that specifies which of the element's properties you want to style.

  • Properties. These are ways in which you can style HTML element. You can choose which properties you want to affect in the rule.

  • Property value. To the right of the property after the colon (:) there is the property value. This chooses one out of many possible appearances for a given property (for example, there are many color values).

Here is the code example for better understanding:

        body {
        background: gray;
        }
        header {
        color: white;
        background: blue;
        }
        p {
        font-size: 18px;
        font-family: monospace;
        line-height: 1.5;
        }

In our example selectors are body, header and paragraph (p) . We will change a style for these selectors.
* Property for body is background and the value is color gray. It means all the background of your page will be gray.
* Properties for header is color and background, with values white and blue. It means that the header background color will become blue and the text in the header - white.
* Properties for paragraph is font-size, font-family and line-height. It means that all the text in your paragraphs will be 18px size, will have monoscpace font and the height between the lines will be 1.5.
